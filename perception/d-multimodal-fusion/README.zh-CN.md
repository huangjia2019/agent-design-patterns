## 预告

多模态融合解决的是输入不只是文字的场景。带图的 PDF、文字夹着结构化表格的
文档、含可 OCR UI 文本的截图。这个模式的核心问题是：什么时候让各模态**并行**
保持独立感知，什么时候把它们**压成**单一文本表示，以及怎么在多模态之间分配
token 预算。

更多见 02-05 讲发布后。
