## Preview

Multi-Modal Fusion handles the case where the input is not just text. PDFs
with figures, structured tables interleaved with prose, screenshots with
OCR-able UI text. The pattern is about deciding *when* to keep modalities
separate (parallel sub-perceptions) versus *when* to flatten them into a
single prose representation, and how to budget tokens across modalities.
